ELLIPSE Portal


Welcome to our ELLIPSE portal project.

This web portal provides visualization and processing tools for data created during the Unmanned Aerial Systems class in spring 2018, held at the Institute for Geoinformatics in the University of Münster under the guidance of Professor Jan Lehmann, Professor Torsten Prinz and Professor Sergey Mukhametov.

The main objective of the project was the development and application of exploratory approaches using several geoespatial technologies to monitor the renaturation process in the Aa River, located in city of Münster, Germany.

The data displayed is a result of extensive joint work. Our class was divided in multiple project groups: Esri Cloud, Sense&Box, Image Analysis, Sensors and Platforms, Lidar, Project Management, Web Portal. More information regarding individual group objectives can be found here.
